Bem-vindo ao Python!

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Foi criada por Guido van Rossum e lançada pela primeira vez em 1991.

Características principais ▼

  • Sintaxe simples e legível: fácil de ler e escrever
  • Interpretada: não precisa ser compilada antes de executar
  • Multiplataforma: funciona em Windows, Linux, Mac
  • Orientada a objetos: suporta programação orientada a objetos
  • Biblioteca extensa: vasta biblioteca padrão
  • Comunidade ativa: grande comunidade de desenvolvedores

Usos comuns do Python ▼

  • Desenvolvimento web (Django, Flask)
  • Ciência de dados (Pandas, NumPy)
  • Inteligência artificial (TensorFlow, PyTorch)
  • Automação de tarefas repetitivas
  • Desenvolvimento de jogos (Pygame)
  • Scripting e automação de sistemas

Exemplo de código Python ▼

print("Olá, Mundo!")

# Variáveis e operações
nome = "Python"
versao = 3.9
